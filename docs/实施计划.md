# BaseAgent 接口
class BaseAgent(ABC):
    @abstractmethod
    async def think(self, problem: str) -> AsyncGenerator[str, None]: 
        """思考阶段，流式输出思考过程"""
        pass
    
    @abstractmethod
    async def code(self, problem: str) -> AsyncGenerator[str, None]: 
        """编码阶段，流式输出代码"""
        pass
    
    @abstractmethod
    async def debate(self, judge_result: JudgeResult) -> DebateArgument:
        """辩驳阶段，生成辩驳论据"""
        pass

**验收标准：**

- 辩驳有轮次限制，不会无限循环
- 裁判系统通过单元测试和集成测试

**评分公式示例：**
```
总分 = 正确性分数 * 0.5 + 效率分数 * 0.3 + 代码质量分数 * 0.2

正确性分数 = (通过测试用例数 / 总测试用例数) * 100
效率分数 = min(100, (参考时间 / 实际时间) * 100)
代码质量分数 = LLM 评审结果（0-100）
```

**状态机示例：**
```python
class ContestStateMachine:
    TRANSITIONS = {
        'waiting': ['running'],
        'running': ['judging'],
        'judging': ['debating', 'finished'],
        'debating': ['judging', 'finished'],
        'finished': []
    }
    
    async def transition_to(self, new_state: str):
        if new_state not in self.TRANSITIONS[self.current_state]:
            raise InvalidTransitionError(...)
        self.current_state = new_state
        await self.broadcast_state_change()
```

## 七、风险管理

| 风险               | 影响 | 概率 | 缓解方案                                               |
| ------------------ | ---- | ---- | ------------------------------------------------------ |
| LLM API 调用成本高 | 高   | 高   | 设置预算限制、支持便宜模型、缓存结果                   |
| Agent 生成恶意代码 | 高   | 中   | Judge0 沙箱完全隔离 + 禁网 + 资源限制                  |
| WebSocket 连接不稳 | 中   | 中   | 自动重连 + 事件补发机制 + 心跳检测                     |
| 辩驳无限循环       | 中   | 低   | 硬性轮次限制（最多 2 轮）+ 超时自动结束                |
| LLM 响应慢         | 中   | 中   | 设置超时限制（最多 5 分钟）+ 给 agent 设定最大思考时间 |
| Judge0 服务宕机    | 高   | 低   | 监控告警 + 自动重启 + 降级为本地 Docker                |
| 数据库性能瓶颈     | 中   | 低   | 索引优化 + 读写分离 + 缓存热数据                       |
